Mes expériences personnelles

Cette liste a été éffectué de l'expérience la plus récente à la plus ancienne
  1. Portfolio
  2. Ce portfolio a été conçu dans le but de me présenter ainsi que de mettre en évidence les différentes compétences que j'ai acquises dans le domaine du développement web.
    Pour le développer, j'ai fait usage de HTML, de CSS et de JavaScript.
    Dans le but de rendre mon site et mon programme plus légers, j'ai utilisé un seul fichier CSS comme base.
    Pour ce qui est du JavaScript est j'ai comme pour le CSS créer un fichier afin que toutes mes fonctions soient au même endroit et utilisable dans tous mes fichiers HTML.
    Dans mon portfolio, chaque page possède son propre fichier HTML avec des noms correspondant à leur contenu.
  3. Bot discord
  4. J'ai il y a quelques mois de cela voulu m'atteler à la conception d'un bot discord dans le but de créer plusieurs interactions avec des utilisateurs, comme la gestion d'expériences en fonction des messages envoyés, un système de modération ou de commandes amusantes.
    J'ai utilisé JavaScript pour coder ce programme afin de pouvoir lier mon programme à une base de données et concevoir des fonctions interactives avec l'utilisateur.
